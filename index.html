FAQ

Questions teams ask before launch

How is this different from an RSS bot?

It groups related articles into stories and keeps multi-source context together, so channels stay readable.

Can I separate topics by channel?

Yes. Default server setup supports up to 3 channels, and each channel can have independent behavior and presets.

Do users have personal controls too?

Yes. `/getnews` and preference controls let individuals run personalized pulls without affecting server defaults.

Are custom feeds supported?

Yes. You can add custom feeds and use owner-side curation tools to monitor feed health and quality.
